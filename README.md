# CodeIgniter 3 to 4 Migration RAG Agent

RAG-powered agent to assist with CI3 to CI4 code migration using LangGraph.

## Project Structure
- `data/raw/` - Original scraped documentation files
- `data/processed/` - Processed chunks
- `src/` - Source code for RAG agent
- `vectordb/` - ChromaDB vector database (not tracked)
